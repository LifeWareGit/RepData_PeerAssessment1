Assignment Part 5: Weekday/Weekend Comparative Analysis
Question: Are there differences in activity patterns between weekdays and weekends?
Answer: Yes, there appears to be more activity (i.e.greater number of steps) during the weekdays in the morning hours but on average more steps are taken on the weekends later in the day.

Task 1: Create a new factor variable in the dataset with two levels - weekday and weekend indicating whether a given date is a weekday or weekend day.

steps_wDaytype <- mutate(steps_filled_dt, 
                       dayType=factor(ifelse(weekdays(date) %in% c("Saturday","Sunday"),
